Formatos principales

Negrita / Destacado

En esta frase tenemos palabras en negrita usando la tag <b> (no semática).
En esta frase tenemos palabras destacadas y en negrita usando la tag <strong> (semática).

Itálica / Énfasis

En esta frase tenemos palabras en itálica usando la tag <i> (no semática).
En esta frase tenemos palabras con énfasis usando la tag <em> (semántica).

Texto resaltado

Podemos crear tambien un texto resaltado usando la tag <mark>.
Aquí hay otro ejemplo de texto resaltado usando otro color

Texto grande y pequeño

En este párrafo estamos creando un texto grande (usando la tag <big>, la cual ya es obsoleta en HTML5) y un texto pequeño (usando la tag <small>).

Texto tachado

Podemos tachar un texto para indicar que debe ser leido pero no considerado.

Texto subrayado

Podemos subrayar un texto para dar énfasis, usando la tag <ins>.
Anteriormente en HTML, se utilizaba la tag <u> (hoy no semántica) para subrayar texto.

Subíndice

Podemos escribir subíndices como en H2O.

Superíndice

Finalmente, podemos escribir superíndices como en x20+3.
