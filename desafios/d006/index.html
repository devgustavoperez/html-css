Tags en HTML

En HTML5 podemos añadir los elementos de la siguiente lista. Pasa el cursor del mouse sobre los términos para descubrir qué tags (etiquetas) debes utilizar en cada caso.

  • Párrafos
  • Títulos
  • Imágenes
  • Abreviaturas
  • Códigos fuente
  • Listas ordenadas (numeración)
  • Listas no ordenadas (viñetas)
